# Workspace-local overrides

Machine-specific config, secrets, and **private persona playbooks** live here. This directory is gitignored except this README and `.gitignore`.

Optional: create `playbooks/<persona>.md` for operator-specific session rituals. Public persona summaries ship from the CoCoder install at `cocoder/personas/playbooks/`. See the install doc `cocoder/personas/playbooks/README-private-operator-pattern.md`.
